Beauty Master

A Modern Beauty Salon App UI built with Flutter

Beauty Master is a beautifully designed Flutter application focused on delivering a smooth and elegant beauty salon experience. The app includes clean layouts, modern UI components, and user friendly flows suitable for salons, spas, and beauty service platforms.

This project is mainly focused on UI, layout consistency, and user experience.

#Getting Started

To run this project locally:

git clone https://github.com/shaukatdj07/Beauty_Master.git
cd Beauty_Master
flutter pub get
flutter run


Make sure Flutter is properly installed on your system.
